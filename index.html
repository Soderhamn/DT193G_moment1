Om den här uppgiften

Den här uppgiften går ut på att skapa en webbplats genom att använda minst två olika CSS-ramverk. Förutom att testa på att använda CSS-ramverken går uppgiften också ut på att diskutera för- och nackdelar med de olika ramverken som används.

För att lösa den här uppgiften har jag använt följande CSS-ramverk:

  • Bootstrap
  • Milligram
  • W3.CSS

Förutom att använda och utvärdera CSS-ramverk ingår det i uppgiften att publicera webbplatsen direkt via GIT istället för att använda en mer traditionell FTP-baserad lösning. För detta ändamål har jag använt GitHub Pages.

GitHub Pages var en lämplig publiceringsplattform för den här uppgiften!
